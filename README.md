## 🎯 Highlighted Projects

### 1. [Quiz Application](https://github.com/yeshant1/Quiz_App) (Microservices Architecture)
* **Description**: A production-ready quiz platform leveraging a microservices pattern by separating responsibilities into isolated services (Authentication, Question, Quiz, Leaderboard, and Payment).
* **Key Architecture & Design**:
  * Decoupled databases: `db_auth`, `db_question`, `db_quiz`, and `db_leaderboard` implementing a database-per-service pattern.
  * Dynamic service registry using **Netflix Eureka** and routing via **Spring Cloud Gateway**.
  * Dynamic leaderboard rankings and premium packages integrated with **Stripe Gateway**.
* **Tech Stack**: Spring Boot, Spring Security, Netflix Eureka, MySQL, Stripe API, React.js.

### 2. [Bank Application](https://github.com/yeshant1) (Production-Grade Microservices System)
* **Description**: Event-driven banking platform featuring core banking operations (Accounts, Loans, Cards) and intelligent virtual assistant harnesses.
* **Key Architecture & Design**:
  * Centralized configuration management using Spring Cloud Config Server.
  * Asynchronous event streams utilizing **Apache Kafka** and **RabbitMQ** for eventual consistency.
  * OAuth2 secure authorization using **Keycloak OIDC**, protected by Spring Security.
  * Observability dashboard metrics monitored using Prometheus and Grafana.
  * Spring AI Agents with MCP functions mapping ledger databases.
* **Tech Stack**: Spring Boot, Spring Security, Spring Cloud, Keycloak, Kafka, Docker, Prometheus, MySQL.

### 3. [E-Commerce Book Application](https://github.com/yeshant1) (Full-Stack GenAI Marketplace)
